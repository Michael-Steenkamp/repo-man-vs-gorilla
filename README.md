# repo-man-vs-gorilla

This repository hosts an Unreal Engine project centered around a "man vs gorilla" theme. It includes a comprehensive set of assets for two distinct characters, various animations, interactive levels, and a collection of sound effects to bring the experience to life.

## Project Structure

The core content is organized within the `Content/` directory:

*   **`Content/Blueprints/`**: Core game logic and interactions.
*   **`Content/Gorilla/`**: All assets related to the gorilla character, including skeletal meshes, animations, and textures.
*   **`Content/Man/`**: All assets related to the human character, including meshes, animations, materials, and textures.
*   **`Content/Input/`**: Input action and mapping configurations.
*   **`Content/Levels/`**: Game maps (`.umap` files) where the action takes place.
*   **`Content/SFX/`**: Sound effects, categorized for footsteps, gorilla grunts, human grunts, and human punches.
*   **`Content/Widgets/`**: User interface elements.

## Getting Started

To explore or develop this project:

1.  Ensure you have a compatible version of Unreal Engine installed.
2.  Clone this repository to your local machine.
3.  Open the `repo-man-vs-gorilla.uproject` file directly using Unreal Engine.
